1. 소수 찾기

문제 설명

1부터 입력받은 숫자 n 사이에 있는 소수의 개수를 반환하는 함수, solution을 만들어 보세요.
소수는 1과 자기 자신으로만 나누어지는 수를 의미합니다. (1은 소수가 아닙니다.)

제한 조건

n은 2이상 1000000이하의 자연수입니다.

입출력 예
n result
10 4
5 3
나의 풀이
function solution(n) {
  const arr = [];
  for (let i = 2; i <= n; i++) {
    arr[i] = i;
  }
  //배열에 반복문 적용
  //[2,3,4,5,6,7,8,9,10]
  for (let i = 0; i <= arr.length; i++) {
    if (arr[i] !== 0) {
      for (let j = arr[i] * 2; j <= arr.length; j += arr[i]) {
        arr[j] = 0;
      }
    }
  }
  return arr.filter(el => el !== 0).length;
}

//소수는 1과 자기 자신으로만 나누어지는 수.
//아리스토텔레스의 체
//2의 배수, 3의 배수, 5의 배수 순서대로 삭제.
  • 어제부터 풀려고 고민하다가 해결책을 못찾아 검색해보니 소수를 구하는 방식 중 **"아리스토텔레의 체"**라는 방식이 있었다. 처음에는 모든 숫자를 나눠주는 방식으로 하려고 했지만, 범위가 늘어날수록 효율성이 떨어지는것을 발견했고 이 공식을 사용하게 되었다.
  • 방식은 2부터 n까지의 숫자를 배열에 나열하고 2부터 자신의 배수를 삭제해가는 방식으로 결국 남는건 소수들만 남게 된다.
  1. 먼저 배열에 2부터 n까지의 숫자를 해당하는 인덱스에 넣는 작업을 해줬다.
  2. 2중 반복문을 사용해서 각 숫자의 배수를 0으로 바꿔주었다.
  3. filter 메서드를 사용해 배열 중 값이 0이 아닌 숫자들만 새로운 배열로 만들어 길이를 반환했다.

2. 수박수박수박수?

문제 설명

길이가 n이고, 수박수박수박수….와 같은 패턴을 유지하는 문자열을 리턴하는 함수, solution을 완성하세요. 예를들어 n이 4이면 수박수박을 리턴하고 3이라면 수박수를 리턴하면 됩니다.

제한 조건

n은 길이 10,000이하인 자연수입니다.

입출력 예
n return
3 수박수
4 수박수박
나의 풀이
function solution(n) {
  const num = Math.floor(n / 2);
  const num2 = n % 2;
  var answer = '';

  for (let i = 0; i < num; i++) {
    answer += '수박';
  }

  if (num2 === 1) {
    answer += '수';
  }
  return answer;
}
  1. num이라는 변수에 n을 2로 나눈 몫을, num2에는 나머지를 저장했다.
  2. 짝수면 ‘박’으로 끝나고 홀수면 ‘수’로 끝나기 때문에 먼저 몫만큼 ‘수박’을 추가하였다.
  3. 홀수일 경우 마지막에 ‘수’만 추가해 반환하도록 작성했다.

3. 문자열을 정수로 바꾸기

문제 설명

문자열 s를 숫자로 변환한 결과를 반환하는 함수, solution을 완성하세요.

제한 조건
  1. s의 길이는 1 이상 5이하입니다.
  2. s의 맨앞에는 부호(+, -)가 올 수 있습니다.
  3. s는 부호와 숫자로만 이루어져있습니다.
  4. s는 0으로 시작하지 않습니다.
입출력 예

예를들어 str이 1234이면 1234를 반환하고, -1234이면 -1234를 반환하면 됩니다.
str은 부호(+,-)와 숫자로만 구성되어 있고, 잘못된 값이 입력되는 경우는 없습니다.

나의 풀이
function solution(s) {
  var answer = Math.abs(s);
  if (s[0] === '-') {
    return -answer;
  }
  return answer;
}

function solution(s) {
  return s / 1;
}
  • 1번 풀이
  1. Math.abs 메서드를 사용해 해당 숫자를 절대값으로 변환했다.
  2. 만약 문자열의 첫번째 값이 -면 음수로 반환했다.
  • 2번 풀이
  • 다른 사람들이 어떻게 풀었는지 보다가 발견한 방식.. Javascript는 (숫자로 구성된)문자열에 곱하기 혹은 나누기를 하면 형변환이 된다.

4. 시저 암호

문제 설명

어떤 문장의 각 알파벳을 일정한 거리만큼 밀어서 다른 알파벳으로 바꾸는 암호화 방식을 시저 암호라고 합니다.
예를 들어 AB는 1만큼 밀면 BC가 되고, 3만큼 밀면 DE가 됩니다. z는 1만큼 밀면 a가 됩니다. 문자열 s와 거리 n을 입력받아 s를 n만큼 민 암호문을 만드는 함수, solution을 완성해 보세요.

제한 조건
  1. 공백은 아무리 밀어도 공백입니다.
  2. s는 알파벳 소문자, 대문자, 공백으로만 이루어져 있습니다.
  3. s의 길이는 8000이하입니다.
  4. n은 1 이상, 25이하인 자연수입니다.

입출력 예

s n result
“AB” 1 “BC”
“z” 1 “a”
“a B z” 4 “e F d”
나의 풀이
function solution(s, n) {
  let arr = s.split('');
  return arr
    .map(el => {
      if (el === ' ') {
        return ' ';
      } else {
        let code = el.charCodeAt();

        if ((code <= 90 && code + n > 90) || (code > 91 && code + n > 122)) {
          return String.fromCharCode(code - 26 + n);
        }
        return String.fromCharCode(code + n);
      }
    })
    .join('');
}
  1. 인자로 들어온 문자열 s를 배열로 변환.
  2. map메서드를 사용해 각 값을 아스키코드로 변환.
  3. 소문자와 대문자에 n값을 더했을때 최대값을 넘어가는 경우에 대해 조건을 만들어 작성.
  4. 반환된 배열을 join 메서드를 사용해 문자열로 합쳐서 반환.

5. 이상한 문자 만들기

문제 설명

문자열 s는 한 개 이상의 단어로 구성되어 있습니다. 각 단어는 하나 이상의 공백문자로 구분되어 있습니다.
각 단어의 짝수번째 알파벳은 대문자로, 홀수번째 알파벳은 소문자로 바꾼 문자열을 리턴하는 함수, solution을 완성하세요.

제한 사항
  1. 문자열 전체의 짝/홀수 인덱스가 아니라, 단어(공백을 기준)별로 짝/홀수 인덱스를 판단해야합니다.
  2. 첫 번째 글자는 0번째 인덱스로 보아 짝수번째 알파벳으로 처리해야 합니다.
입출력 예
s return
try hello world TrY HeLlO WoRlD
나의 풀이
function solution(s) {
  let arr = s.toLowerCase().split(' ');
  for (let i = 0; i < arr.length; i++) {
    let word = '';
    for (let j = 0; j < arr[i].length; j++) {
      if (j % 2 === 0) {
        word += arr[i][j].toUpperCase();
      } else {
        word += arr[i][j];
      }
    }
    arr[i] = word;
  }
  return arr.join(' ');
}
  1. s 문자열을 모두 소문자로 변환 후 단어별로 나눠서 새로운 배열을 생성했다.
  2. 2중 반복문을 돌려 각 단어별로 인덱스의 홀수, 짝수 여부에 따라 대문자로 변경하거나 소문자를 그대로 word라는 변수에 넣었다.
  3. 하나의 단어에 대한 변환 작업이 끝나면 배열에 들어가있는 단어를 변경했다.
  4. 변환이 완료 된 배열을 join 함수를 사용해 공백을 추가해 문자열로 반환했다.

6. 정수 내림차순으로 배치하기

문제 설명

함수 solution은 정수 n을 매개변수로 입력받습니다. n의 각 자릿수를 큰것부터 작은 순으로 정렬한 새로운 정수를 리턴해주세요. 예를들어 n이 118372면 873211을 리턴하면 됩니다.

제한 조건

n은 1이상 8000000000 이하인 자연수입니다.

입출력 예
n return
118372 873211
나의 풀이
function solution(n) {
  return parseInt(
    n
      .toString()
      .split('')
      .sort((a, b) => b - a)
      .join(''),
  );
}
  1. n을 문자로 변환해 split 메서드를 통해 배열로 만들었다.
  2. sort메서드를 사용해 내림차순으로 순서를 만들고 join 메서드를 사용했다.
  3. 마지막으로 parseInt 메서드를 사용해 숫자로 변환해 리턴했다.
