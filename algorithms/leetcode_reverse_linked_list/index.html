[Leetcode] Reverse Linked List

Javascript를 제대로 이해하고 사용하는 개발자가 되기 위해 공부중입니다.
오늘은 Leetcode에서 Reverse Linked List를 풀어봤습니다.

문제

Reverse a singly linked list.

Example:
Input: 1->2->3->4->5->NULL
Output: 5->4->3->2->1->NULL

나의 풀이

var reverseList = function (head) {
  if (!head) {
    return null;
  }
  let prev = null;
  let cur = head;
  let next = cur.next;

  while (cur) {
    cur.next = prev;
    prev = cur;
    cur = next;
    if (next) {
      next = next.next;
    }
  }
  return prev;
};
